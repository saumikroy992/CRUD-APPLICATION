# CRUD-APPLICATION

This is a simple Node.js CRUD application using Express and MongoDB. It allows you to Create, Read, Update, and Delete product records.

## 🚀 Features

- Create a new product
- Get all products
- Get a product by ID
- Update a product
- Delete a product

## 🧰 Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- Nodemon
